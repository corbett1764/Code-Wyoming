The Shapes of CSS

All of the below use only a single HTML element. Any kind of CSS goes, as long as it's supported in at least one browser.

Square

#square { 
    width: 100px; 
    height: 100px; 
    background: red; }

Rectangle

#rectangle { 
    width: 200px; 
    height: 100px;
    background: red; }

Circle

#circle {
	width: 100px;
	height: 100px;
	background: red;
	border-radius: 50px;}

Putting in a closing line is always a popuar thing to do to help fill in space.
